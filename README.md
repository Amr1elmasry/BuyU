# BuyU is a E-commerce Mobiles Website 
- Its Asp.net Core mvc with Identity and roles 
- My main goal was to focus on Backend so i have used For Backend :
  1. asp.net Core V6
  2. SqlServer
- And for front :
  1. html
  2. css
  3. javascript & Jquery
  4. Bootstrap

- AdminRole Has Access To control products, users, and roles
